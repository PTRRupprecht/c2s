# c2s

Tools for the prediction of spike trains from calcium traces.

## Requirements

* Python >= 2.7.0
* [cmt](https://github.com/lucastheis/cmt) >= 0.5.0
* NumPy >= 1.6.1
* SciPy >= 0.13.0
